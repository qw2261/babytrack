## 四、API 接口测试

### 4.1 创建事件

```bash
# 创建洗澡事件
curl -X POST http://localhost:5000/api/events \
  -H "Content-Type: application/json" \
  -d '{"type":"bath","start_time":"2026-05-16T10:00"}'

# 创建睡眠事件
curl -X POST http://localhost:5000/api/events \
  -H "Content-Type: application/json" \
  -d '{"type":"sleep","start_time":"2026-05-16T11:00"}'
```

**结果**: ✅ 创建成功，返回事件ID

### 4.2 查询事件

```bash
curl http://localhost:5000/api/events?date=2026-05-16
```

**结果**: ✅ 返回当天所有事件列表

### 4.3 更新事件(结束)

```bash
# 结束洗澡事件
curl -X PUT http://localhost:5000/api/events/{bath_id} \
  -H "Content-Type: application/json" \
  -d '{"end_time":"2026-05-16T10:30"}'
```

**结果**: ✅ 更新成功

### 4.4 汇总数据

```bash
curl http://localhost:5000/api/summary/2026-05-16
```

**结果**: ✅ 返回当日各项统计
